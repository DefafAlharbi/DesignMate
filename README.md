# DesignMate AI

An AI-powered UI/UX design assistant built with a Multi-Agent Architecture.

## Features

- Generate UI designs from text prompts.
- Review uploaded UI designs.
- Detect UI/UX issues.
- Provide AI-powered design recommendations.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- OpenRouter API
- Gemini 2.5 Flash
- Claude code
